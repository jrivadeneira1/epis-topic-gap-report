# EPIS Topic Gap Analyzer Report

This repository contains a static prototype report generated from the EPIS Topic Gap Analyzer project.

The prototype analyzes EPIS publication PDFs, extracts text, scores topic categories, filters candidate gaps by working-group relevance, and generates a readable HTML report.

The results should be treated as candidate topic gaps for human review, not as final editorial conclusions.

## Files

- index.html: main report
- EPIS_Topic_Gap_Analyzer_Explainer_Revised.pdf: short project explanation
